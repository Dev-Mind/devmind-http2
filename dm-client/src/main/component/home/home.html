Performance web Spring Boot

Cette application exemple sert de support à ma présentation sur l'amélioration des performances d'une webapp Spring Boot . Le but est de voir les améliorations que vous pouvez faire côté serveur ou côté présentation pour que votre application soit performante. Quand on parle performance nous pouvons dissocier les aspects liés à la sécurité et veux à la rapidité (2 critères utilisés aujourd'hui dans le calcul du page rank Google).

L'exemple fonctionnel utilisé est la conférence Mix-IT
